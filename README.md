# FireForce
누군가가 부탁해서 만들게 된 불꽃 소방대 스크립트
### 실행 환경

* Java 21
* Paper(기반의 소프트웨어) 1.21.8

### 적용법(서버 여는 방법은 아시죠?)

1. [스크립트 깃허브 Releases](https://github.com/SkriptLang/Skript/releases)에서 가장 최신 버전 Skript-xxx.jar을 받아 서버 폴더의 'plugins/' 폴더에 옮겨주세요.
2. 서버를 껐다가 다시 시작해주세요.
3. [현재 이 페이지](https://github.com/1cowoo/FireForce/)에서 초록색 "Code" 버튼을 누른후 "Download Zip"을 누릅니다.

### 주의점 
1. 유튜브와 같은 플랫폼에서 사용할시 이 깃허브 링크 혹은 [유튜브 채널](https://youtube.com/@KOWOO코우)을 영상 설명창에 써주세요.
